Add a click event listener on the Add new element button to execute the addElement function and:

  • add a new li child element of the list-container ul list with the day name of: Monday.
  • NOTE: this should be done using the addEventListener method on the Add new element button.
  • the li element should have the class of weekday-item
  • you can use the className property to assign the class.
  • the new li element should be added using the insertAdjacentElement() method on the first child element of the list-container parent node, with the position of beforebegin so that it is added as the first child element.
  • the new element should only be added once if you click the button more than one time.
