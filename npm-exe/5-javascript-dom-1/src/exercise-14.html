Instructions

When you are finished execute in the browser console the function validateExercise14() the check your solution.

Add a click event listener on the Add list button to execute the addList function to:

  • create a ul list based on the week days in the days array with an li element for each day.
  • the ul list parent element should have a class of list-container.
  • each li child element should have a class of weekday-item and the text content of the day from the array (using a loop to create the elements).
  • the list should be added as a child element of the ex-wrapper div.
  • the list should only be added once.
  • HINT: use the childElementCount property to check if the parent has no child elements.
