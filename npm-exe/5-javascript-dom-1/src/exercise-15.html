When you are finished execute in the browser console the function validateExercise15() the check your solution.

Add a click event listener on the Clone list button to execute the cloneList function to:

  • create a deep clone of the list-container ul list.
  • the cloned list should be added as a child element of the ex-wrapper div after the existing list.
  • the cloned list should have an id of new_list.
  • HINT: you will need to set the deep parameter of the cloneElement() method to true to copy all the children not just the parent node.
