When you are finished execute in the browser console the function validateExercise17() the check your solution.

Add mouse event listeners on the #change_classes button to add or remove its class of hovered.

  • add the class hovered to the className property of the Change classes button when it receives the mouseover event (the mouse cursor is positioned over the button).
  • remove the class hovered from the className property of the button when it receives the mouseout event (the mouse cursor leaves the button).
