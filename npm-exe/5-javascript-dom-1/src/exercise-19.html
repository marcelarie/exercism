Add a click event listener on the image to execute the onImageClick function to change the src attribute of the image to the one in the newSrc variable.

You will have to get a reference to the image using the event.target property of the event parameter of the onImageClick function.

event.target.src = "my-src"

event.target will be the image that was clicked.
