Complete the code of the handleSubmit function to output the form values as paragraphs when the form is submitted.

  • add an event listener of submit using addEventListener on the form element.
  • prevent the default action of submitting a form with event.preventDefault()
  • access the values of the firstName, lastName and email fields to create 2 p element paragraphs with the contents:
    • 'Name:' + empty space + value of firstName field + empty space + value of the lastName field.
    • Example: Name: Ana Maria
    • 'Email:' + empty space + value of email field.
    • Example: Email: ana@mail.com
  • append the paragraphs as the last child elements of the form.

The form fields can be accessed either by using querySelector or any method to query DOM elements with JavaScript, or by using the event parameter of the function event.target.elements.fieldNameInTheForm.value

Example: event.target.elements.firstName.value will have the value of the input field with a name of: firstName.
